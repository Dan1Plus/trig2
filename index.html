Описание работы сайта

Данный сайт может помочь Вам в решении различных уравнений графическим способом. Чтобы воспользоваться сайтом, необходимо иметь хотя бы одно уравнение, чтобы построить его график. Если уравнений несколько, то функционал сайта позволяет представить их всех на одной координатной плоскости. Те координаты точек по оси Ох, где графики пересекаются, будут являться корнями уравнения, состоящего из заданных Вами функций. Увидеть значения координат можно в правом верхнем углу, при перемещении курсора в плоскости координат: сначала пишется числовое значение, потом значение в π (для удобства решения тригонометрических уравнений). Сайт представляет возможность самостоятельно выбрать масштабы координатной плоскости. Для этого нужно в специальные поля вписать свои значения: напротив x_left - минимальное значение оси ОХ; напротив x_right - максимальное значение оси ОХ; напротив y_min - минимальное значение оси ОY; напротив y_max - максимальное значение оси ОY (дробные значения пишутся через точку). Вы также можете выбрать цвет графика, который собираетесь построить (доступно восемь цветов).

Далее необходимо вписать свою функцию в соответствующее поле (снизу от графического блока). Все дробные значения необходимо писать через точку, операцию умножения нужно прописывать с помощью знака "*", деления - "/". Модуль x: abs(x); синус х: sin(x); косинус x: cos(x); тангенс х: tan(x); котангенс x: ctan(x); квадратный корень из х: sqrt(x); экспонента: exp(x); логарифм числа х по основанию a: log(a,x); число х в степени a: step(x,a).

При нажатии на кнопку "Построить график", будет нарисован график введённой функции в заданной по размерам системе координат. Можно рисовать сколько угодно графиков. Если захочется изменить масштаб или удалить графики, то это можно осуществить, предварительно нажав кнопку "Стереть всё".

Данный инструмент вполне позволяет решать различные уравнения школьной программы, а также с его помощью можно проверить правильность аналитических решений уравнений.
